# Demand Forecast Studio v2 · 수요예측 시스템

월별 시계열 데이터를 업로드하면 **13종 시계열·ML·Foundation 모형**과 앙상블로 향후 수요(금액·건수)를 예측하고, 인터랙티브 차트와 Word 보고서를 제공하는 웹 애플리케이션입니다.

## ✨ v2 새로운 기능

- 🌗 **라이트/다크 테마 토글** — 우측 상단 버튼
- 💱 **통화 단위 전환** — KRW(원) / USD(달러)
- 📊 **수입+수출 동시 분석** — 차트에 두 시리즈 겹쳐서 표시
- 🔢 **금액 + 건수** — 단일/복합 동시 예측 (이중 Y축)
- 🔍 **데이터 범위 선택** — 전자상거래 vs 전체 무역
- ⏱ **관측 기간 듀얼 슬라이더** — 학습용 데이터 구간 직접 조절
- 📅 **예측 기간 자유 조절** — 1~36개월 슬라이더 + 빠른 선택(3/6/12/24)
- 🎯 **신뢰수준 80/95/99%** 선택
- 📈 **11개 예측 모형** (5개 카테고리)
- ❓ **모형 카드 hover 툴팁** — 적합 상황·장단점 즉시 확인
- 📚 **전체 모형 카탈로그 모달** — "전체 모형 설명 보기" 버튼

## 🤖 적용된 13개 예측 모형

| 카테고리 | 모형 | 설명 |
|---------|------|------|
| **기본** | 이동평균 (MA) | 최근 N개월 평균 |
| | 계절성 단순 (SN) | 전년 동월값 |
| **지수평활** | Simple Exp Smoothing (SES) | 단순 지수평활 |
| | Holt's Linear Trend | 레벨+추세 |
| | Holt-Winters (HW) | 레벨+추세+계절성 |
| **ARIMA** | AR(p) | 자기회귀 |
| | ARIMA(p,d,q) | 비계절 정석 |
| | SARIMA | 계절 ARIMA |
| **분해** | STL Decomposition | 추세·계절·잔차 분해 |
| | (Prophet) | *서버 필요 - 준비중* |
| **머신러닝** | Random Forest | 30 trees, lag 특징 |
| | Gradient Boosting (XGB) | 50 rounds |
| **딥러닝** | (LSTM, DeepAR, TFT) | *서버/GPU 필요 - 준비중* |
| **앙상블** | 자동 평균 | 2개 이상 선택 시 자동 |

## 🚀 GitHub Pages 배포

### 가장 간단한 방법

1. 새 저장소 생성 (Public)
2. 이 폴더의 모든 파일을 업로드
3. **Settings → Pages**에서 Source를 `Deploy from a branch / main / / (root)`로 설정
4. 1~2분 후 `https://USERNAME.github.io/REPO/`에서 접속

### 필수 파일

```
index.html      # 메인 페이지
app.js          # 모든 로직
theme.css       # 라이트/다크 테마
sample.js       # 샘플 데이터 (선택)
.nojekyll       # GitHub Pages용
```

## 📁 데이터 형식

수입/수출 시트 각각에 다음 컬럼이 있어야 합니다:

| 연도 | 월 | (전자상거래) 금액 | (전자상거래) 건수 | (전체) 금액 | (전체) 건수 |

- **시트명**에 "수입" / "수출" 포함 필수
- 컬럼명에 "전자상거래" / "전체"와 "금액" / "건수" 포함
