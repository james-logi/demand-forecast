# Demand Forecast Studio · 수요예측 시스템

월별 시계열 데이터를 업로드하면 5종 시계열 모형(이동평균·선형추세·지수평활·Holt-Winters·계절성단순)과 앙상블로 향후 수요를 예측하고, 인터랙티브 차트와 Word 보고서를 제공하는 웹 애플리케이션입니다.

## 데모

배포 후 `https://<your-username>.github.io/<repo-name>/` 에서 바로 사용할 수 있습니다.

## 주요 기능

- **엑셀 업로드 (.xlsx, .xls, .csv)** — 시트와 컬럼을 자동 인식하여 변수 선택 가능
- **5종 예측 모형 + 앙상블** — 단일 또는 다중 선택, 모형별 동적 해설 자동 생성
- **예측 기간 선택** — 3 / 6 / 12 / 24개월
- **출력 형태 선택** — 점 예측 / 예상 범위(신뢰구간) / 둘 다 동시 표시
- **신뢰수준 조정** — 80% / 95%
- **고급 파라미터** — 이동평균 윈도우, 지수평활 α, Holt-Winters β·γ를 슬라이더로 조정
- **주식 차트 스타일 인터랙션** — 마우스 호버 시 크로스헤어와 정확한 값 표시
- **Word 보고서 출력** — 한 번의 클릭으로 분석 결과를 docx로 다운로드
- **100% 클라이언트 사이드** — 서버 없이 브라우저에서 모든 처리, 데이터 외부 전송 없음

## GitHub Pages 배포 방법

### 방법 1 — 가장 간단 (수동)

1. 이 저장소를 fork하거나 새 저장소에 파일들을 업로드
2. GitHub 저장소 → **Settings** → **Pages**
3. **Source** 를 `Deploy from a branch` 로 설정
4. **Branch** 를 `main`, 폴더는 `/ (root)` 로 선택 후 Save
5. 1~2분 후 `https://<username>.github.io/<repo>/` 에서 접속 가능

### 방법 2 — GitHub Actions (자동 배포)

`.github/workflows/deploy.yml` 파일이 이미 포함되어 있어, main 브랜치에 푸시하면 자동 배포됩니다.

1. 저장소 → **Settings** → **Pages** → **Source** 를 `GitHub Actions` 로 설정
2. main 브랜치에 푸시하면 자동 빌드·배포

## 데이터 형식

엑셀 시트는 다음 형식이어야 합니다:

| 연도 | 월 | (분석 변수1) | (분석 변수2) | ... |
|------|----|-------------|-------------|-----|
| 2014 | 1  | 84,577,963  | 869,808     |     |
| 2014 | 2  | 79,079,212  | 796,398     |     |

- `연도` (또는 `Year`), `월` (또는 `Month`) 컬럼은 반드시 포함되어야 합니다
- 그 외 숫자 컬럼은 자동으로 분석 가능한 변수로 인식됩니다
- 여러 시트가 있는 경우 시트 선택 드롭다운으로 전환 가능

## 샘플 데이터

상단의 **샘플** 버튼을 누르면 BANDTrass 무역통계의 전자상거래 수입·수출 데이터(2014~2026)로 즉시 체험할 수 있습니다.

## 적용된 예측 모형

| 모형 | 특징 | 적합한 데이터 |
|-----|------|--------------|
| 이동평균 (MA) | 최근 N개월 평균을 외삽 | 추세·계절성이 약한 평탄한 데이터 |
| 선형 추세 (LT) | 회귀선을 외삽 | 명확한 장기 성장 추세 |
| 지수평활 (ES) | 최근값 가중 평균 | 단기 변동 반영이 필요할 때 |
| Holt-Winters (HW) | 레벨+추세+계절성 | 월별 패턴이 강한 시계열 |
| 계절성 단순 (SN) | 전년 동월값 사용 | 강한 계절성, 약한 추세 |
| 앙상블 | 위 모형들의 평균 | 일반적으로 가장 안정적 |

## 기술 스택

- 순수 HTML / CSS / Vanilla JavaScript (빌드 없음)
- Tailwind CSS (CDN)
- SheetJS (xlsx 파싱)
- docx.js (Word 문서 생성)
- 자체 구현 SVG 차트 엔진 (라이브러리 의존 없는 인터랙티브 차트)
